It’s a sunny winter afternoon (-4 degrees in the morning), perfect, lets get on with this.

Build the site#

  1. copy and backup the while website folder locally
  2. check hugo version, run “hugo version” in terminal
  3. I need to update hugo, run “brew upgrade hugo” in terminal
  4. go to folder 1 level above - parent folder
  5. run “hugo new site huatuostudio.github.io –force –format yaml” in terminal
  6. download PaperMod theme and extract the folder from zip file
  7. move the folder inside of the themems folder
  8. run “git init” in terminal

One of the major decision point to use the theme is that it’s multilingual support. Look Scanned Blog has a good post on how to modify the language selector.

Configurations#

adding this to hugo.yaml to make

params:
  displayFullLangName: true
